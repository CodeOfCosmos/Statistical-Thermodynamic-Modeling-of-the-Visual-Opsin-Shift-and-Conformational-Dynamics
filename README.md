# Statistical-Thermodynamic-Modeling-of-the-Visual-Opsin-Shift-and-Conformational-Dynamics
a discrete, two-state thermodynamic model to quantify the macromolecular free energy transition from inactive Rhodopsin to active Metarhodopsin II.Formulated and numerically integrated canonical partition functions Z_cis and Z_trans in Python over a defined reaction coordinate (ξ) to track macromolecular population distributions.Simulated the biological "opsin shift" by incorporating a 22.0 k_{B} steric work function W_push representing the mechanical push of all-trans retinal, demonstrating a complete spontaneous inversion of the net Helmholtz Free Energy (Δ F) from a stable positive baseline to a favorable negative driving force.
